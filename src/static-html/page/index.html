ページタイトル

これは固定ページの汎用テンプレートです。プライバシーポリシーや行動規範など、タイトルと文章が主体のページに使い回せます。この本文エリア（.c-prose）に文章を書くと、見出し・段落・リスト・リンクなどに自動でスタイルが当たります。

見出し（h2）

大見出しはセクションの区切りに使います。段落の文章はこのように表示されます。長い文章でも読みやすい行間・文字サイズに調整されています。

小見出し（h3）

中見出しは h2 の中の項目分けに使います。箇条書きは次のように表示されます。

  • 箇条書きの項目その1
  • 箇条書きの項目その2
  • 箇条書きの項目その3

番号付きリスト

  1. 手順の1つめ
  2. 手順の2つめ
  3. 手順の3つめ

リンクについて

本文中のリンクはこのように表示されます。外部サイトへのリンクや関連ページへの案内に使えます。詳しくはトップページをご覧ください。

※このページはテンプレートのため、内容はダミーです。実際のページを作るときは、このファイルを複製してタイトルと本文を差し替えてください。
